## gTTS

Este código converte texto em audio utilizando o módulo python gTTS.

## Preparativos

Para executar este código, verifique se possui o módulo `gTTS` instalado.

```py
pip install gTTS
```

Após isso, poderá utilizar o comando

```py
py main.py
```

para inicializar a conversão!

## Execução

Lembre-se de alterar o texto inserido em `text.txt` para o texto que deseja converter para voz.

## Opções

Segue uma lista de idiomas permitidos:

```py
lang='pt'
lang='es'
lang='ru'
lang='fr'
```
